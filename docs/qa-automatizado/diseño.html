Estructura básica del proyecto NUnit​

El paquete de Microsoft.Playwright.NUnit permite implementar la interfaz PageTest en los archivos de pruebas, lo cual habilita el uso del objeto Page para localizar los diferentes elementos (locators) y navegar a través de los enlaces de la UI. Desde el explorador de pruebas se ejecutan los bloques de código con el decorador [Test] y se muestran los resultados de las pruebas.

Configuración del proyecto​

Se puede custommizar la configuración básica de playwright sobrescribiendo las opciones del objeto browser.

La siguiente configuración permite grabar toda la sesión de los tests ejecutados a través de capturas de vídeo y de capturas de pantalla.

using Microsoft.Playwright.NUnit;
using Microsoft.Playwright;

namespace PlaywrightTestPlanWithDotNet;

public class Tests : PageTest

{
    // Configuración inicial del contexto
    [SetUp]
    public async Task Setup()
    {
        // Inicio de trazabilidad
        await Context.Tracing.StartAsync(new()
        {
            Screenshots = true,
            Snapshots = true,
            Sources = true
        });
    }
    // Configuración de captura de vídeo
    public override BrowserNewContextOptions ContextOptions()
    {
        var options = base.ContextOptions() ?? new();
        options.RecordVideoDir = "videos";
        return options;
    }
    [Test]
    public async Task Test1()
    {
        //Act
        await Page.GotoAsync("https://playwright.dev");
        await Page.GetByText("GET STARTED").ClickAsync();
        //Assert
        Assert.Pass();
    }

    //Configuración de cierre
    [TearDown]
    public async Task TearDown()
    {
        // Fin trazabilidad y registro en trace.zip
        if (Context?.Tracing != null)
        {
            var tracePath = "trace.zip";
            await Context.Tracing.StopAsync(new() { Path = tracePath });
            TestContext.AddTestAttachment(tracePath);
        }
        // Adjuntar el vídeo
        if (Page?.Video != null)
        {
            var videoPath = await Page.Video.PathAsync();
            TestContext.AddTestAttachment(videoPath);
        }
    }
}

Los archivos generados se guardarn en \PlaywrightTestPlanWithDotNet\bin\Debug\net8.0. El archivo trace.zip se puede visualizar como reporte a través del siguiente comando:

pwsh bin/Debug/netX/playwright.ps1 show-trace https://example.com/trace.zip
